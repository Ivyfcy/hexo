引用类型

2019年02月18日 |
JavaScript
| Object Array JavaScript

[TOC]

Object类型

  • 创建Object实例的方式有两种
    • new操作符后跟Object构造函数
    • 对象字面量
  • 使用对象字面量语法时,如果留空其花括号,则只包含默认属性和方法 eg:var person={}//与new Object()相同

获取属性的值

  • 点表示法（.）：属性名用一个标识符来表示，标识符必须直接出现在JavaScript程序中，他们不是数据类型，程序无法修改他们
  • 方括号表示法（[表达式]）:表达式必须返回字符串（或返回一个可以转换为字符串的值）；属性名通过字符串来表示，字符串是JavaScript的数据类型，在程序运行时可以修改和创建它们.eg: customer["adress"+i]

    如果属性名中包含会导致语法错误的字符，或者属性名使用的是关键字或保留字，也可以使用方括 号表示法 eg:person["first name"] = "Nicholas"

Array类型

  • 创建数组的基本方式有两种
    • 使用new Array 构造函数
    • 使用数组字面量表示法
  • 使用new Array 构造函数时:
    • 如果传入一个 数值`var colors=new Array(20),这个数值表示该数组的长度;如果是其他类型的参数`,则会创建包含那个值的只有一个项的数组

      注意:数组的 length 属性很有特点——它不是只读的(通过设置这个属性，可以从数组的末尾移 除项或向数组中添加新项)

检测数组: Array.isArray()方法

转换方法

  • 所有对象都具有toLocaleString()、toString()和 valueOf()方法
  • toString():返回由数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串
  • valueOf():返回的还是数组

数组继承的 toLocaleString()、toString()和 valueOf()方法，在默认情况下都会以逗号分隔的字符串的形式返回数组项;使用 join()方法(只用于数组,返回字符串)，则可以使用不同的分隔符来构建这个字符串join("||")

如果数组中的某一项的值是 null 或者 undefined，那么该值在 join()、 toLocaleString()、toString()和 valueOf()方法返回的结果中以空字符串表示。

栈方法

  • 后进先出的数据结构(最新添加的项最早被移除):为数组提供 push()和pop()方法来实现类似栈的行为
  • 而栈中项的插入(叫做推入)和移除(叫做 弹出)，只发生在一个位置——栈的顶部
  • push():接收任意数量的参数,逐个添加到数组的末尾,并返回修改后数组的长度
  • pop():从数组末尾移除最后一项,减少数组的length长度,返回移除的项

队列方法

  • 队列数据结构的访问规则是先进先出;使用shift()和 push()方法可以像使用队列一样使用数组。
  • push():向数组末端添加项
  • shift():移除数组中的第一个项并返回该项，同时将数组长度减 1
  • 队列在列表的末端添加项，从列表的前端移除项

unshift()方法 它能在数组前端添加任意个项并返回新数组的长度(unshift()与 shift()的用途相反)

使用 unshift()和 pop()方法，可以 从相反的方向来模拟队列，即在数组的前端添加项，从数组末端移除项

重排序方法

  • reverse()和 sort()—改变原数组
  • sort()方法会调用每个数组项的 toString()转型方法，然后比较得到的字符串,所以sort()方法可以接收一个比较函数作为参数,
  • 比较函数接收两个参数
    • 正序:element1排element2之前，函数返回一个负数（return -1）
    • 倒序:element1排element2之后，函数返回一个正数（return 1）
    • element1，element2相等，返回0
  • 对于数值类型或者其 valueOf()方法会返回数值类型的对象类型，可以使用一个更简单的比较函数,即只要用第二个值减去第一个值`function compare(value1, value2){
    return value2 - value1;
    }`

操作方法

  • concat()方法(原数组不变):连接产生新的数组
    • 没有传递参数时,只是复制当前数组并返回副本
    • 有参数(无论是不是数组)时,都会将每一项添加到结果数组中
  • slice()方法(原数组不变):返回一个数组的子集（新数组)—截取数组
    • slice(start,end)返回数组不包括结束处的元素;
    • 参数可以为负数,用数组的长度加上该负数来确定相应的位置slice(-2,-1)该数组有5项,则slice(3,4)
    • 如果只有一个参数,从start处开始到数组尾部
  • splice()方法:删除,插入,替换元素( 改变原数组)
    • 返回一个数组(删除的项),该数组是包含从原始数组中删除的项(如果没有删除项,则返回空数组)
    • splice(start，element-to-delete，value1，value2…)
      • start:一个数字（数组索引值）：删除第一项的位置
      • element-to-delete（可选）从start处开始要删除元素的个数，if省略，则从start处开始到数组的最后 全删
      • value1…插入数组的值

位置的方法

  • indexOf()–从数组的开头开始向后找，lastIndexOf()–从数组的末尾开始向前找，
  • 参数:接收两个参数（或一个参数)—要查找的项；查找起点位置的索引(可选)
  • 返回值:如果找到了返回查找的项在数组中的位置，没有找到返回-1；
  • 查找时的比较严格等于(===)
  • eg:indexOf()结合for循环可以实现数组去重功能
    arr.forEach(function(v,i,arr){ var bool=arr.indexOf(v,i+1); if(bool===-1){ result.push(v) } })

迭代方法

  • 为数组定义了5个迭代方法(不会修改原数组的值)
  • 每个方法都接收两个参数(函数，函数的作用域对象(影响this的值)(可选));
  • 传入的函数接收三个参数:数组项的值,该项在数组中的位置和数组对象本身
    • forEach():没有返回值,本质上和for循环迭代数组一样
      • arr.forEach(function(item,index,array){ // 执行某些操作}) 如果对数组array操作,原数组改变
    • map():让数组通过某种计算产生一个新数组
      • 返回一个数组var newArr=arr.map(function(item,index,array){ return item*2})
    • filter():筛选出数组中符合条件的项,组成新数组
      • 返回符合条件的项组成新数组var newArr=arr.filter(function(item,index,array){ return (item>2)})
    • every():检测数组每一项是否符合条件,对每一项都返回true,则结果返回true
      • 用于查询数组中的项是否满足某个条件;每一项都返回true,这个方法才返回true,否则返回falsevar everyArr=arr.every(function(item,index,array){ return (item>2)})
    • some():检测数组是否有些项符合条件,某一项返回true,则结果返回true
      • 只要传入的函数对数组中的某一项返回 true，就会返回 truesomeArr=arr.some(function(item,index,array){ return (item>2)})

归并方法

  • 归并数组的方法:reduce(),reduceRight()
  • reduce():接收一个函数作为累加器，数组中的每个值(从左到右)开始缩减，最终为一个值；传入的函数接收四个参数:前一个值，当前值，项的索引，数组对象。
    • 求数组中所有值之和的操作var values = [1,2,3,4,5]; var sum = values.reduce(function(prev, cur, index, array){ return prev + cur; }); alert(sum); //15
  • reduceRight():则从数组的最后一项开始，向前遍历到第一项
    • reduceRight()与reduce()的作用类似，只不过方向相反而已
