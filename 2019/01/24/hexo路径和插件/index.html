自定义路径

  • 在根目录下_config.yml文件中配置

  • 在scaffolds文件夹下的post.md文件中添加urlname:属性

  • 最后在写的文章里添加你想要的路径名

Front-matter写法

  • Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量
  • 除了 YAML 外，你也可以使用 JSON 来编写 Front-matter，只要将 — 代换成 ;;; 即可。
